# React Todo List

這是一個使用 **React** 和 **Vite** 製作的 Todo List 專案</br>
具備 **現代化** **neon** 卡片 UI，支援任務的新增、編輯、刪除與完成狀態切換

## 功能特色

- 新增待辦事項
- 編輯、刪除任務
- 點擊任務可切換完成狀態
- 現代卡片式設計，支援響應式
- 使用 React Hooks 管理狀態

## 專案結構

```
src/
	components/
		CreateForm.jsx   // 新增任務表單
		EditForm.jsx     // 編輯任務表單
		Todo.jsx         // 單一任務元件
		TodoWrapper.jsx  // 任務列表與邏輯
	App.jsx
	App.css
```

## 如何啟動

1. 安裝依賴
   ```
   npm install
   ```
2. 啟動開發伺服器
   ```
   npm run dev
   ```

## 技術棧

- React 19
- Vite
- React Icons
- CSS3
